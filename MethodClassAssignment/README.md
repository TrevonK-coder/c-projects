# Method Class Assignment

A simple C# console application demonstrating class instantiation, method definitions taking parameters, positional arguments, and named arguments.

## Requirements

1. Create a class (`MathOperations`).
2. In that class, create a `void` method (`PerformMathAndDisplay`) that takes two integers as parameters. Have the method do a math operation on the first integer and display the second integer to the screen.
3. In the `Main()` method of the console app, instantiate the class.
4. Call the method in the class, passing in two numbers (positional arguments).
5. Call the method in the class, specifying the parameters by name (named arguments).
6. Fully commented lines/blocks of code explaining the implementation.

## How to Build and Run

### Using .NET SDK

1. Open a terminal in this directory.
2. Run:
   ```bash
   dotnet run
   ```

### Using legacy .NET Framework Compiler (`csc.exe`)

1. Open a terminal in this directory.
2. Compile:
   ```cmd
   C:\Windows\Microsoft.NET\Framework64\v4.0.30319\csc.exe Program.cs
   ```
3. Run the compiled executable:
   ```cmd
   .\Program.exe
   ```
