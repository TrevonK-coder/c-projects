# Datetime Assignment

A C# console application demonstrating date and time operations using C#'s `System.DateTime` class.

## Features & Requirements Met

1. **Current Time Print**: Prints the current system date and time using `DateTime.Now`.
2. **User Input Integration**: Prompts the user to input a number (representing hours).
3. **Future Time Calculation**: Computes and displays the exact time it will be in X hours using `DateTime.AddHours()`.
4. **Detailed Comments**: Fully documented with detailed developer comments explaining each syntax structure.

## How to Build and Run

### Using the .NET SDK

1. Navigate to this directory in your terminal.
2. Execute:
   ```bash
   dotnet run
   ```

### Using legacy .NET Framework Compiler (`csc.exe`)

1. Open a terminal in this directory.
2. Compile:
   ```cmd
   C:\Windows\Microsoft.NET\Framework64\v4.0.30319\csc.exe Program.cs
   ```
3. Run the compiled executable:
   ```cmd
   .\Program.exe
   ```
